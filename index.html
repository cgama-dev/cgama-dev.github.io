Projeto DevRuning

Essa aplicação é um gerenciador de corridas. Nesse projeto além do gerenciamento das corridas é abordado alguns conceitos de autenticação e autorização, utilizando JsonWebToken. Na área administrativa podendo controlar os usuários logados e suas corridas. Uma área restrita onde os usuários logados cadastram suas corridas. Nessa aplicação as tecnologias utilizadas foram: NodeJS (express), Sqlite3, Jwt para API BackEnd, no FrontEnd é utilizado React, React-route, Redux-Saga, MomentJs
